# poc-java-i18n-pluralization

Proof of Concept: i18n with Java and pluralization compatible to phraseapp.com

&nbsp;

### Build and Run

```bash
./gradlw build
java -jar ./build/libs/poc-i18n-0.1.0.jar

# or run directly
./gradlw run
```
